# Options API

A production-style REST API for querying and filtering options market data in real time. Built with FastAPI, SQLAlchemy, and Python — designed as a solo project to demonstrate backend API development with financial data.

## Features
- Real-time options chain data by ticker and expiry
- Filter options by strike price, volume, open interest, and IV
- Historical options data with date range queries
- API key authentication and rate limiting
- Auto-generated interactive docs at `/docs`

## Tech Stack
Python · FastAPI · SQLAlchemy · Pydantic · APScheduler · SQLite · Docker
